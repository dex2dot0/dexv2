## Tech Stack

### Main Frontend

- [SST](sst.dev) - Serverless Stack Toolkit V3
- [SvelteKit](https://kit.svelte.dev/) - Svelte Awesomeness
- [Skeleton](https://www.skeleton.dev/) - UI Toolkit for Svelte using [TailwindCSS](https://tailwindcss.com/)
